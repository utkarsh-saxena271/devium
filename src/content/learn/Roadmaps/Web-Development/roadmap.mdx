## Phase I

### Week I
- **HTML**
    - tags vs attributes
    - boilerplate code
    - heading tags (h1 to h6)
    - tags like - p, b, br, i, u, img, etc.
    - anchor tag
    - lists
    - tables
    - semantic tags
    - div and span
- **CSS**
    - basic properties - height, width, background-color, etc.
    - box model
    - inline and block elements
    - flex and grid
    - media queries
    - keyframes

Build a basic web page -> navbar, hero section, and some sections like FAQs, social proof, etc., followed by a footer.

**JavaScript Basics** -> basic syntax, data types, keywords, variables

- control flow (if, else if, else / switch-case)
- loops (for, while, forEach, etc.)

**JavaScript Core** -> arrays, objects, functions (with properties), and DOM manipulation

**Async JavaScript** -> core async nature, Web APIs, etc.

Now let's talk about advanced stuff, like upgrading yourself to:
- TypeScript
- Next.js
- PostgreSQL or any other database
- ORMs like Prisma
- real-time communication using - WebSockets, socket.io, WebRTC, etc.
- Redis
- payment integration

Then my suggestion would be to first create a full-stack MERN app using whatever you have learned, and while building that project, also learn about payment integration.

After this, you should start learning things in this sequence:

- TypeScript 
- PostgreSQL
- Prisma
- socket.io / WebSockets (I'd say learn both)
- Redis

And not only keep learning, but keep building simple and complex projects on the go, because that will make you better. If you do so and reach this point, you'll no longer need a roadmap to move ahead.
